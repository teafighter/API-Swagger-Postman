Учебное веб приложение для работы со списком студентов.
В рамках этой домашней работы отработаны следующие знания и навыки:
- Построение структуры веб-приложения на Spring
- Подключен Swagger, опробованы HTTP-запросы с помощью Postman
- Подключена база данных для хранения информации
- Изучены потоки данных и применены для работы с файлами (аватарки, хранящиеся в БД)
- Опробовано написание SQL запросов
- Опробована работа со схемами баз данных (затронуто написание миграций и создание индексов)
- Подключено логирование действия в приложении
- Базово изучена многопоточность (параллельные стримы и потоки)
